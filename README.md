# global-scope

<!-- [![build status][1]][2]

[![browser support][3]][4] -->

Require global variables

## Example

```js
var global = require("global-scope")
var document = require("global-scope/document")
var window = require("global-scope/window")
```

## Installation

`npm install global-scope`
